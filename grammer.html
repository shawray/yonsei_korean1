语法문법

语法

问候 인사

입니다

  • 附于体词之后，充当谓语。用来表示事物的身份、状态或指定某个事物。
  • 저는 학생입니다 我是学生

은/는

  • 体词的助词，表示句子所陈述的主体或要讨论的话题。体词以辅音结尾用은，以元音结尾用는。
  • 有时表示对比或强调。
  • 可以和所有助词结合。当은/는加在主格助词或者宾格助词后面时，主格助词이/가或者宾格助词을/를要省略，但如果附于其他助词之后，该助词不能省略。
  • 저는 한국사람입니다 我是韩国人

입니까

  • 입니다的疑问形式。
  • 이분이 어느 나라 사람입니까? 这位是哪国人?

이/가 아니다

  • 和体词连用，是이다的否定形式。如果体词以辅音结尾，用이 아니다，以元音结尾用가 아니다。
  • 저는 일본 사람이 아닙니다 我不是日本人

습니다/ㅂ니다

  • 和谓词词干连用，用于向对方解释或描述一个具体的动作或状态。如以辅音结尾用습니다，以元音结尾用ㅂ니다 。
  • 학교에 갑니다 我去学校

습니까/ㅂ니까

  • 和谓词词干连用，表示对具体动作或状态的疑问。如以辅音结尾用습니까，以元音结尾用ㅂ니까。
  • 무엇을 합니까? 你在做什么?

学校和家 학교와 집

이/가

  • 用于体词后面的主格助词，表示主语。如果体词末音节以辅音结尾用이，以元音结尾用가。
  • 人称代词나，너，저，及疑问代词누구与가结合后，分别变为：내가，네가，제가，구가。
  • 이것이 책상입니다 这是桌子
  • 제가 김영수입니다 我是金英秀

이/그/저

  • 이用在名词前，表示所谈论的人或事物离说话人很近。或者表示说话人和听话人正在说的事实。
  • 그用在名词前，表示所谈论的人或事物离听话人近，离说话人远。也可以表示说话人和听话人之前说过的或者都知道的人或事物。
  • 저用在名词前，表示所谈论的人或事物离说话人和听话人都远。
  • 이 책이 사전입이까? 这本书是词典吗?
  • 그 책은 교과서입니다. 那本书是课本.
  • 저 건물이 우리 학교입니다. 那栋楼是我们学校

도

  • 用于体词后面的助词，表示列举相同或相似的事物。与主格助词이/가、宾格助词을/를连用时，主格助词、宾格助词要省略。
  • 当与에或에서连用表示场所或位置时，에、에서不能省略。
  • 책상이 있습니다. 의자도 있습니다. 시계도 있습니다. 有桌子,也有椅子,也有钟表.

있다/없다

  • 있다表示有、在，其否定形式是없다。
  • 如果主语是需要表示尊敬的人，用계시다、안 계시다。
  • 친구가 있습니다 我有朋友
  • 돈이 없습니다 我没有钱

에

  • 和体词连用的助词，在这里表示主语的存在处所。通常和있다/없다连用。
  • 설악산이 강원도에 있습니다 雪岳山在江原道

하고

  • 和体词连用的助词，连接两个或两个以上的体词。意思和그리고相近。
  • 가방하고 구두를 삽니다 我买包和皮鞋

家人和朋友 가족과 친구

을/를

  • 附于体词后面的宾格助词，表示句子的宾语。如果体词以辅音结尾用을，以原因结尾用를。
  • 점심에는 보통 김밥을 먹습니다 我中午一般吃紫菜包饭

으시/시

  • 尊称词尾，一般用于谓词词干之后，表示对主语的尊敬。如果词干以辅音结尾用으시，以元音结尾用시。此后缀通常和敬语主格助词께서一起使用。
  • 有些词在变成敬语时不加으시/시，而需要换成自身带有尊敬意义的词。자다 -> 주무시다, 먹다 -> 잡주시다, 있다 -> 게시다
  • 어머니께서 가십니다 妈妈去

数词

  • 韩语中表示数量的时候，数词后面要加单位名词。需要注意的是，当数词하나,둘,셋,넷在单位名词前出现时，要变成한,두,세,네的形式。
  • 사람 한 면/두 면 一个人/两个人

고1

  • 连接词尾，用于谓词词干和体词谓词形之后，表示罗列两种或两种以上的动作、状态或事实。
  • 내일은 춥고 비가 오겠습니다 明天很冷，还会下雨

에서

  • 助词，用于表示处所的名词之后，表示动作进行的场所。
  • 도서관에서 공부합니다 在图书馆学习

饮食 음식

에 가다/오다

  • 当表示主体的存在处所的助词에和趋向动词가다/오다连用时，에前面的处所名词就是动词的目的地。
  • 친구가 우리 집에 옵니다 朋友来我家

을까요/ㄹ까요?1

  • 用于谓词词干后面的疑问式终结词尾，表示向对方提议一起做某事或者询问意见。如果谓词词干以元音结尾用ㄹ까요，以辅音结尾用을까요。
  • 점심을 같이 먹을까요？ 我们中午一起吃饭好吗？

읍시다/ㅂ시다

  • 用于动词词干后面的终结词尾。表示提议一起做某事或者接受对方的请求。如果动词词干以元音结尾用ㅂ시다，以辅音结尾用읍시다。
  • 한국 음식을 먹읍시다　我们吃韩国料理吧

지 않다

  • 用于谓词词干后面,表示否定. 该句型不能用于命令句或共动句.
  • 내일은 학교에 가지 않습니다 明天不去学校

ㄹ特殊音变

  • 以ㄹ结尾的谓词词干和以ㄴ,ㅂ,ㅅ开头的词尾连接时,ㄹ脱落.
  • 살다: 살 + ㅂ니다 -> 삽니다

고 싶다

  • 用于动词词干之后,表示说话人的愿望.当主语是第三人称时,用고 싶어하다.
  • 고향에 가고 싶습니다 我想去我的家乡

겠1

  • 和动词词干连用,表示说话人的意志. 在疑问句中,겠用来询问听话人的意图.
  • 저는 비빔밥을 먹겠습니다 我要吃拌饭

으십시오/십시오

  • 和动词词干连用的终结词尾,表示命令或嘱托.如果动词词干以元音结尾用십시오,以辅音结尾用으십시오.
  • 내일 7시에 오십시오 请明天7点来

을까요/ㄹ까요?2

  • 和动词词干连用的终结词尾,表示向对方询问关于某事的意见或想法.如果动词词干以元音结尾用ㄹ까요,以辅音结尾用을까요.
  • 언제 다시 전화할까요? 我什么时候再给你打电话好呢?

一天的生活 하루 생활

까지

  • 用于体词后面的助词,表示时间或地点的终点以及顺序的末尾.
  • 12시에서 1시까지 점심시간입니다 午餐时间是12点到1点

어요/아요/여요

  • 非格式体终结词尾,用于谓词词干和体词谓词形后,在表示尊敬的同时,带有亲切柔和的语气.该词尾可以用于陈述句,疑问句,命令句或共动句.

  • 格式体用于书面和正式场合,非格式体用于口语.

  • | | 格式体 | 非格式体 | | :----: | :-------------: | :-------------: | | 陈述句 | 습니다/ㅂ니다 | 어요/아요/여요 | | 疑问句 | 습니까/ㅂ니까 | 어요/아요/여요? | | 命令句 | 으십시오/십시오 | 으세요/세요 | | 共动句 | 읍시다/ㅂ시다 | 어요/아요/여요 |

  • 如果谓词词干以아,오,야结尾用아요,以其他元音结尾用어요,以하다结尾用여요,通常缩写为해요.

  • 如果谓词词干以元音结尾,同时词尾又以元音开始,两个元音就会结合在一起,有时缩写,有时省略.

  • 省略: 아 + 아 -> 아, 어 + 어 -> 어, 으 + 어 -> 어

  • 缩写: 오 + 아 -> 와, 우 + 어 -> 워, 이 + 어 -> 여, 하 + 여 -> 해

지요

  • 用于谓词词干和体词谓词形后面的终结词尾,表示说话人对所提问题已有所了解,只是希望得到对方确认.
  • 내일 학교에 가지요? 明天去学校吧?

으세요/세요

  • 这种终结词尾是在어요/아요/여요的前面加上尊敬词尾으시/시得来的,可以用于陈述句,疑问句,命令句,共动句.如果前面的词干以元音结尾用세요,以辅音结尾用으세요.
  • 어떤 음식이 좋으세요? 你喜欢什么样的食物?

에

  • 和表示时间的体词连用,表示某种动作或者状况发生的时间.但是不与언제(什么时候),오늘(今天),내일(明天),어제(昨天),그저께(前天),모레(后天)等词连用.
  • 9시에 학교에 갑니다 我9点去学校

부터 ~ 까지

  • 用于体词后,부터表示某一范围的起点,까지表示某一范围的终点,相当于汉语的"从...到..."
  • 월요일부터 금요일까지 공부합니다 我从九点到一点学习

고2

  • 连接词尾,和动词词干连用,表示动作发生的先后顺序,此时고不和过去时的词尾았/었/였连用.整个句子的时态是由고后面的动词决定的.
  • 숙제를 하고 잡니다 我做完作业再睡觉

았/었/였

  • 和谓词词干以及体词谓词形连用,表示过去时.如果词干以元音아,오结尾用았,以其他元音结尾用었.如果以하다结尾,用였,组成하였습니다 했습니다.
  • 점심을 먹었습니다 我吃过午饭了

ㅂ的特殊音变

  • 以ㅂ结尾的部分谓词和以元音开头的词尾连接时,ㅂ变成우,如덥다,춥다. 需要注意的是,돕다,곱다这两个词,当后面连接以어,아开头的的词尾时, ㅂ变成오, 连接以으开头的词尾时,ㅂ变成우.而像입다, 잡다, 씹다, 좁다, 넓다等谓词属于规则变化,故不遵守此规则.
  • 어제는 날씨가 아주 더웠습니다 昨天天气非常热

购物 물건 사기

으러/러

  • 连接动词词干的连接词尾,通常和移动性动词오다 ,가다连用,表示移动的目的或意图.如果动词词干以元音结尾用러,以辅音结尾用으러.
  • 영화를 보러 갈까요? 我们去看电影怎么样?

과/와

  • 连接助词,用于体词和体词的连接,表示并列关系.如果体词以辅音结尾用과,以元音结尾用와.在口语中,可以用하고代替.
  • 책상 위에 시게와 책이 있습니다 桌子上有钟表和书

지만

  • 用于谓词词干和体词谓词形后面的连接词尾,表示转折.
  • 값은 비싸지만 물간이 좋아요 价钱虽然贵了些,但是东西质量很好.

定语词尾은/ㄴ

  • 用于形容词词干和体词谓词形之后,充当修饰成分,表示事物现在的性质,状态.如果词干以辅音结尾用은,以元音结尾用ㄴ.
  • 키가 큰 사람이 동생입니다 个子高的是弟弟

ㅎ的特殊音变

  • 以ㅎ结尾的形容词词干和以"ㄴ,ㄹ,ㅁ,ㅂ,元音"开头的词尾连接时, ㅎ脱落.如果形容词词干后面接으开头的词尾时,ㅎ和词尾的으都脱落.如果形容词词干后面接어/아/여开头的词尾时,ㅎ脱落,词干的元音和词尾的元音相结合成애. 需要注意的是,形容词좋다属于规则变化,故不遵守此规则.
  • 어떤 색을 좋어합니까? 你喜欢什么颜色?

어/아/여 주다

  • 用于动词词干后面,表示为某人做某事.如果听话人是长辈或上级,用어/아/여 드리다.
  • 이름을 써 주세요 请写一下您的名字

单位名词

  • 数事物时要在数词后面加上单位名词. 根据单位名词的不同,数词的读法也不同(固有词,汉字词).
  • 맥주 두 병에 얼마예요? 两瓶啤酒多少钱?

에게

  • 助词,用于表示人或者动物的体词后面,表示宾语涉及的对象,通常和주다 보내다 가르치다 빌력주다这类动词连用.在口语中,常用한테,如果对象是需要尊敬的人时,要将에게/한테换成敬语体께.
  • 저에게 한국말을 가르쳐 주세요 请教我韩语

定语词尾는, 은/ㄴ, 을/ㄹ

  • 用于动词词干后面,修饰后面的体词.如果该动作已发生,用은/ㄴ;如果是正在进行中,用는;如果动作将要发生或表示对将来的推测,打算,用을/ㄹ.
  • 지금 일하는 회사가 어디예요? 你在哪家公司上班?

交通 교통

으로/로1

  • 用于处所名词后面的助词,表示移动方位或目的地.后面通常接가다 오다 돌아가다 돌아오다 等动词.名词以辅音结尾用으로,以元音或ㄹ结尾用로.
  • 사무실로 갑니다 我去办公室

아서/어서/여서1

  • 用于谓词词干和体词谓词形后面的连接词尾,表示后面的动作或状态在某种情况下发生或形成.当词干末音节元音为아,오时用아서,末音节元音为其他元音时用어서,以하다结尾时用여서,一般缩写称해서.
  • 집에 가서 점심을 먹습니다 我回家吃饭

으로/로2

  • 助词,用于体词后面,表示方式,原因,资格.
  • 젓가락으로 먹습니다 我用筷子吃饭

에서 ~ 까지

  • 用于表示时间,处所的体词后,에서表示起点,까지表示终点,类似于"从...到..."
  • 서울에서 부산까지 기차로 5시간 걸립니다 从首尔到釜山坐火车需要5个小时

아서/어서/여서2

  • 用于谓词词干后面的连接词尾,表示动作的连贯或原因.表示原因时,终结词尾不能用命令句或共动句.另外,时制词尾았/었/였和겠不能用于아서/어서/여서前面.
  • 헌국이 좋아서 한국에 왔습니다 我喜欢韩国,所以来到韩国

으니까/니까

  • 和谓词词干或体词谓词形连用的连接词尾,表示前句是后句的原因或进行前面的动作后才知道后面的状况.如果前面音节以元音结尾用니까,以辅音结尾用으니까.
  • 오늘은 일이 많으니까 내일 만납시다 今天有很多事情要做,我们明天见吧

지 말다

  • 和动词词干连用,表示对行为的禁止,类似于汉语的"不要""别".是命令句和共动句的否定形式.
  • 건물 안에서 담배를 피우지 마십시오 请不要在楼离抽烟.

ㄷ的特殊音变

  • 以ㄷ结尾的部分谓词词干和以元音开头的词尾连接时,ㄷ变成ㄹ.需要注意的是,받다 믿다 얻다 닫다等词属于规则变化,故不遵守此规则.
  • 듣 + 어서 -> 들어서
  • 음악을 들으니까 기분이 좋아요 听了音乐,心情很好

르的特殊音变

  • 词干末音节以르结尾的大部分谓词,如果后面接어/아开头的词尾,那么르中的元音'ㅡ'脱落,同时在르前面的音节上添加收音ㄹ.
  • 모르 + 아서 -> 몰라서

电话 전화

을게요/ㄹ게요

  • 用于动词词干后面的终结词尾,表示说话人的意志,誓言或决定.
  • 이따가 전화할게요 等一会儿我会给你打电话的.

이나/나

  • 用于体词后面的助词,表示从罗列的两个或两个以上的事物中选择一个.
  • 주말에는 농구나 축구를 합시다 我们周末一起打篮球或踢足球吧.

는데요, 은데요/ㄴ데요

  • 用于体词谓词形和谓词词干后面的终结词尾,表示说话人叙述某一情况并期待得到对方的回应.动词词干后面用는데요.形容词词干以元音结尾用ㄴ데요,以辅音结尾用은데요,体词谓词形后面用ㄴ데요.
  • 이 시험은 좀 어려운데요 这次考试有点难

에게서/한테서

  • 助词,用于活动体体词后面,表示某动作或状态的出发点.通常与"배우다 듣다 받다 빌리다"这类动词一起使用,意思相当于으로부터(从).
  • 친한 친구에게서 돈을 빌렸어요 我从好朋友那儿借了钱

은면/면

  • 用于谓词词干和体词谓词形的连接词尾,表示前句是后句的假设条件.如果词干末音节以元音结尾用면,以辅音结尾用은면.
  • 내일 날씨가 좋으명 설악산에 갑시다 如果明天天气好的话,我们一起去爬雪岳山吧

을/ㄹ 거예요

  • 用于谓词词干和体词谓词形后面.如果前面音节以辅音结尾用을 거예요,以元音结尾用ㄹ 거예요
  • 如果主语是第一人称或第二人称,表示说话人的意志或者打算.
  • 저녁에 불고기를 만들 거예요 晚上我要做烤肉
  • 如果主语是第三人称,表示说话人的推测或猜想.
  • 영수 씨가 요즘 좀 바쁠 거예요 英秀这几天会很忙

만

  • 用于名词或部分副词后面的助词,表示"仅仅,只"
  • 10분만 자고 싶어요 我只想睡十分钟

天气和季节 날씨와 졔절

는데,은데/ㄴ데

  • 用于谓词词干和体词谓词形后面的连接词尾,表示对立,转折. 动词词干后面用는데,形容词词干以辅音结尾用은데,以元音结尾用ㄴ데,体词谓词形后面用ㄴ데.
  • 저는 농구를 좋아하는데 친구들은 야구를 좋아합니다 我喜欢篮球,但是朋友们喜欢棒球

을/ㄹ 수 있다

  • 和谓词词干及体词谓词形连用,用来表示有做某事的能力或有某种可能性. 其否定形式是 을/ㄹ 수 없다. 如果前面词干以辅音结尾用을 수 있다,以元音结尾用ㄹ 수 있다.
  • 나도 한구말을 할 수 있습니다 我也会说韩语

은/ㄴ 후에

  • 用于动词词干后面,表示前一个动作完成之后,后一个动作又开始. 如果动词词干以辅音结尾用은 후에, 以元音结尾用ㄴ 후에.
  • 학교를 졸업한 후에 뭘 하시겠어요? 你毕业以后想做什么?

겠2

  • 用于谓词词干和体词谓词形后面的词缀,表示说话人的推测.
  • 주말이라서 교통이 복잡하겠어요 因为是周末,所以交通一定很拥堵.

보다

  • 和体词连用的助词,用于两个事物之间的比较.보다前面是被比较的对象.
  • 지하철이 버스보다 더 빨라요 地铁比公交快

을/ㄹ 것 같다

  • 用于谓词词干后,表示对将要发生的事情的推测. 如果词干以辅音结尾用을 것 같다,以元音结尾用ㄹ 것 같다. 正在进行的动作用는 것 같다. 已完成的动作和状态用ㄴ/은 것 같다.
  • 이번 시험이 어려울 것 같아요 这次考试看起来会很难

는군요/군요

  • 用于谓词词干和体词谓词形后面的终结词尾,表示感叹,评价或判断. 动词词干后面用는군요, 形容词词干和体词谓词形后面用군요, 었/았/였和겠后面用군요.
  • 매운 음식을 잘 잡수시는군요 你真能吃辣

고 있다

  • 用于动词词干后,表示动作正在进行,属于现在进行时.
  • 지금 뭐 하고 있어요? 你现在干什么呢?

节假日 휴일과 방학

으려고/려고 하다

  • 用于动词词干后,表示计划或意图. 如果动词词干以辅音结尾用으려고, 以元音结尾用려고.
  • 내일 대사관에 가려고 합니다 我打算明天去大使馆

동안

  • 用于表示时间段的名词后面, 意思是"期间"
  • 방학 동안 여행을 많이 하고 싶어요 假期我想去很多地方旅行

을/ㄹ 때

  • 用于谓词词干和体词谓词型之后, 表示动作状态发生的时间. 如果词干以辅音结尾用을 때, 以元音结尾用ㄹ 때. 如果表示某个动作状态已经结束, 用过去时词尾었을/았을/였을 때.
  • 名词后面直接用때
  • 저역을 먹을 때 손님이 오셨어요 我们吃晚饭的时候,客人来了
  • 시험 때 考试时

중에서

  • 用于表示各种事物/对象的体词后面, 意思是"...当中"
  • 한국 음식 중에서 무엇이 제일 맛이 있어요? 你觉得韩国料理中什么最好吃?

에 ~ 쯤

  • 에一般接在单位名词或者可以作为单位的名词后面, 表示基准. 后面常和쯤连用,表示对数量,时间或地点的大致估计.
  • 하루에 8시간쯤 자요 我每天差不多睡8个小时

못

  • 主要用于动词前面的副词, 表示"不能够","没有这种能力"
  • 매운 음식을 못 먹어요 我不能吃辣

기 전에

  • 用于谓词词干后, 表示一个动作发生在另外一个动作之前.
  • 名词后面,直接用전에
  • 밥을 먹기 전에 손을 씻어요 吃饭前先洗手

지 못하다

  • 用于谓词词干后,表示"不能够","没有这种能力"
  • 술은 한 방울도 마시지 못해요 我一点酒也不能喝
